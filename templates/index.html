Welcome to Hockey Drills DB

Hockey Drills DB is a site to help coaches shared hockey drills among their fellow colleagues.

Project developed for Udacity Full Stack Web Developer Nanodegree by Eduardo Paz Orduña. The purpose of this website is to allow users view/publish/edit hockey drills. You can login with Google Authentication System.

The site is organized in categories and drills.

The site has been developed using.

Vagrant

Flask

SQLAlchemy

Google Authentication
